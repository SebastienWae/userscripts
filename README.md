# userscripts
A bunch of userscripts, mostly vibe coded.

## Scripts

| Script | Description | Install |
|--------|-------------|---------|
| [Copy Article as Markdown](src/copy_as_md.js) | Copy article content to clipboard as markdown using Turndown | [Install](https://raw.githubusercontent.com/SebastienWae/userscripts/main/src/copy_as_md.js) |
| [Google Search - Maps Button](src/google_search_map_button.js) | Adds a Maps button to Google Search results and makes map images clickable | [Install](https://raw.githubusercontent.com/SebastienWae/userscripts/main/src/google_search_map_button.js) |
| [Hacker News - Dark Mode](src/hn_dark.js) | Dark theme for HN without white flash (works with BFCache) | [Install](https://raw.githubusercontent.com/SebastienWae/userscripts/main/src/hn_dark.js) |
| [Meteoblue - Auto Cookies](src/meteoblue_cookies.js) | Set cookies to disable adblock protection and define settings | [Install](https://raw.githubusercontent.com/SebastienWae/userscripts/main/src/meteoblue_cookies.js) |
| [System Font Override](src/system_font.js) | Force system font on any website (per-domain toggle) | [Install](https://raw.githubusercontent.com/SebastienWae/userscripts/main/src/system_font.js) |
| [X/Twitter - Enable Timeline Ranking](src/x_popular_timeline.js) | Rewrites HomeTimeline (GET) and HomeLatestTimeline (POST) to enable ranking | [Install](https://raw.githubusercontent.com/SebastienWae/userscripts/main/src/x_popular_timeline.js) |
